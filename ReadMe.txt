Project Title : Movie Explorer

This program is that Users can add or find information about movies in movie files. 
Also, users are able to check the movie information in the movie file at random as the number of movies they want.

The date : May 18, 2020

The author : Sangmean Hong, Aiden(Gookjin) Kim

Users can run this program by running a file with a "jar" extension on the command prompt. 
Users should type "java -jar filename.jar" on the command prompt.
